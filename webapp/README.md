# web-app


## Install NODE.js and npu

[more info](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)



## Install python requirements
```
pip install -r requirements.txt
```

## Project setup
```
npm install
```

## Compiles for production
```
npm run build
```

## Run app.py
```
export DEVICE_ID=1
python3 python app.py
```
